
NOTE: import is `nmOps` (not `nmClipboard` as in junk) — onContextMenuCommand lives there now.

**`op=suppress`** — sets $suppress_confirm AND $nmDialogs::suppress_error_dialog together
(they travel as a pair per todo.md). Also accepts `outcome=accept|reject` to set
$nmDialogs::suppress_outcome.

**`op=refresh`** — calls $pane->refresh() to reload navMate.db from disk. Defaults
to `panel=database` if no panel param given.

**`select=key1,key2,...`** — _doSelect calls _walkSelect (recursive). Handles:
- UUID keys (waypoints, groups, routes, tracks)
- `header:groups`, `header:routes`, `header:tracks` (E80 header nodes)
- `my_waypoints` (E80 ungrouped WPs node)
- `root` (tree root)
- `rp:ROUTE_UUID:WP_UUID` (route_point keys -- wires to `_getNodeKey`)
- Auto-expands collapsed branches (DUMMY placeholder), records expanded keys
  so _doFire can collapse them again after command runs

**`cmd=N`** — _doFire calls `onContextMenuCommand($cmd_id, $panel_name, $rc, $tree, @nodes)`.
Collapses auto-expanded branches after command returns.

**`right_click=key`** — defaults to first key in select if not specified.
