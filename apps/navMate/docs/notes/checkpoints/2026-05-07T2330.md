### winMain.pm — updated

Added `use nmDialogs;` import (after `use nmTest;`).

Added `showError` override:
```perl
sub showError
{
    my ($this, $msg) = @_;
    return if $nmDialogs::suppress_error_dialog;
    $this->SUPER::showError($msg);
}
```
When `$suppress_error_dialog` is set (test mode), blocking modal is skipped.
Ring buffer already has the ERROR message from `Pub::Utils::error()` before showError is reached.
